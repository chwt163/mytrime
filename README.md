# 常规布局：
逗号键右滑---主题
句号键左滑---颜色
空格长按或上滑---中英切换
删除键上滑---清空候选

# 七列布局：
逗号键右滑---主题
句号键左滑---颜色
空格上滑或长按---中英切换
删除键上滑---清空候选
“#123”键上滑或长按---符号键盘
